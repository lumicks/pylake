Examples
========

This section contains sample scripts to analyze data from different applications. You can find application notes for all these segments `here <https://lumicks.com/applications/all/>`_.

For all of the examples, it is assumed that the following lines precede any other code::

    import numpy as np
    import matplotlib.pyplot as plt

    import lumicks.pylake as lk
